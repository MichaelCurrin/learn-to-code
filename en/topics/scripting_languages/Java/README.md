# Java


## Hello world app


1. Create file `hello.java`.
    ```java
    public class HelloWorld {

        public static void main(String[] args) {
            System.out.println("Hello, world!");
        }

    }
    ```
2. Compile.
    ```sh
    $ javac hello.java
    ```
3. Execute the output binary.
    ```sh
    $ ./hello
    Hello, world!
    ```
