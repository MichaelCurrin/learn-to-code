# Java


## Hello world app


1. Create file `HelloWorld.java`.
    ```java
    public class HelloWorld {

        public static void main(String[] args) {
            System.out.println("Hello, world!");
        }

    }
    ```
2. Compile.
    ```sh
    $ javac HelloWorld.java
    ```
3. Execute the output binary.
    ```sh
    $ ./HelloWorld
    ```
