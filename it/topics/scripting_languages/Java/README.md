# Java


## Ciao Mondo app


1. Crea il file `ciao.java`.
    ```java
    public class CiaoMondo {

        public static void main(String[] args) {
            System.out.println("Ciao, mondo!");
        }

    }
    ```
2. Compilalo.
    ```sh
    $ javac ciao.java
    ```
3. Esegui il file di output in formato binario gi√† compilato in precedenza.
    ```sh
    $ ./ciao
    Ciao, mondo!
    ```
